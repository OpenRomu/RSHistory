# RSHistory

**The Unwritten History of RomuStrike**

This repository is a work in progress tracking and archiving RomuStrike
versions scattered around the internet. It contains a YAML "database" of
every known version with its date from publicly available information online.

If you're interested of the *Written* part of the history, you may find it
here: https://romustrike.fr/archives/index.html
